### Flow 1: Creating a Reason Code
1. Click "Add Reason Code" button
2. Modal opens with empty form
3. Enter code (auto-uppercased)
4. Enter description
5. Select one or more categories
6. Toggle active status if needed
7. Click "Create"
8. Success message appears
9. Modal closes
10. New code appears in list

### Flow 2: Editing a Reason Code
1. Click edit icon (pencil) on a row
2. Modal opens with pre-filled data
3. Modify any fields
4. Click "Update"
5. Success message appears
6. Modal closes
7. Updated code appears in list

### Flow 3: Deleting a Reason Code
1. Click delete icon (trash) on a row
2. Confirmation dialog appears
3. Review the code being deleted
4. Click "Delete" to confirm or "Cancel" to abort
5. If confirmed, success message appears
6. Code removed from list
